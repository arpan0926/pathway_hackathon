# 5-Minute Pitch Script

## Introduction (30 seconds)
- Problem statement
- Team introduction
- Solution overview

## Live Demo (3 minutes)
1. Show GPS simulator running
2. Display live vehicle movement on map
3. Demonstrate ETA calculation and updates
4. Trigger and show one alert
5. Ask chatbot one query and show response

## Technical Architecture (1 minute)
- Pathway pipeline for real-time processing
- FastAPI backend
- Streamlit dashboard
- RAG-powered chatbot

## Q&A Preparation (30 seconds)
- Key differentiators
- Scalability approach
- Future enhancements

## Notes
- Keep it simple and clear
- Focus on working features
- Emphasize real-time capabilities
- Be ready to explain ETA logic
