# Shared Resources

This folder contains code and configurations shared across all members.

## Contents
- `schemas.py` - Common data models
- `constants.py` - Shared constants
- `utils.py` - Utility functions
- `config.py` - Global configuration

## Usage
Import shared resources in your module:
```python
from shared.schemas import ShipmentData
from shared.constants import ALERT_TYPES
```
